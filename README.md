# Titanic EDA Project

## Objective
Perform Exploratory Data Analysis (EDA) on the Titanic dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Features
- Data Cleaning
- Missing Value Handling
- Age Distribution Visualization
- Survival Count Analysis
- Passenger Class Distribution
